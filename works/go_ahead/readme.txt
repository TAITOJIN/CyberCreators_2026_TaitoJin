◆ゲーム作品について
//----------------------------------
・ツール...GUI で扱いやすい
・物理挙動
・マルチスレッドによる Now Loading (ローディング画面)
・サウンド...距離に応じて音が聞こえたり、聞こえなかったりします
・見た目...ライティングを変えました
・外部ファイル...ini や json ファイルなどを駆使しました


//----------------------------------
◆基本操作
//----------------------------------
・キーボード操作の場合
【ENTER】          : 画面遷移
【Ｗ, Ｓ, Ａ, Ｄ】 : 前方，後方，左，右へ進む

・コントローラーの場合
【左スティック】: キャラクターの移動
【Ａ / Ｂ】     : ジャンプ
