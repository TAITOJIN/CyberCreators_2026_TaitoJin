//----------------------------------
◆ゲーム作品について
//----------------------------------
・GUI で扱いやすいツールを作成！
・物理挙動の実装！
・マルチスレッドによる Now Loading (ローディング画面)
・没入感を深めるサウンド！距離に応じて音が聞こえたり、聞こえなかったりします
・ライティングを変え、明るい世界観に！見た目もこだわります
・外部ファイルを駆使しました。ini や json 、xlsx など


//----------------------------------
◆基本操作
//----------------------------------
・キーボード操作の場合
【ENTER】          : 画面遷移
【Ｗ, Ｓ, Ａ, Ｄ】 : 前方，後方，左，右へ進む

・コントローラーの場合
【左スティック】: キャラクターの移動
【Ａ / Ｂ】     : ジャンプ
